\# Entrenador Miguel



Aplicación PWA personal para el seguimiento de entrenamientos de fuerza.



\## Estado



Versión candidata: \*\*v1.0.0-RC1\*\*



\## Objetivo



Aplicación diseñada para uso personal durante entrenamientos de fuerza, optimizada para dispositivos móviles y funcionamiento como Progressive Web App (PWA).



\## Características principales



\- Plan de entrenamiento estructurado por fases.

\- Motor de sesiones.

\- Recuperación de sesiones interrumpidas.

\- Temporizadores de trabajo y descanso.

\- Historial de sesiones.

\- Funcionamiento offline mediante Service Worker.

\- Instalación como PWA.
